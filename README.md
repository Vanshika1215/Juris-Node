⚖️ Juris Node

AI-powered legal contract analysis platform that helps users upload contracts, detect risky clauses, generate summaries, and download professional PDF reports.

Built with React + FastAPI + Gemini AI.

---

# 🚀 Features

## 🔐 Authentication
- Email/password login & registration
- Google OAuth login
- Protected routes with session management

## 📄 Contract Upload
- Upload digital PDFs
- OCR support for scanned documents
- Drag-and-drop upload flow

## 🤖 AI Contract Analysis
Gemini AI analyzes contracts and classifies clauses into:
- Critical
- Caution
- Standard

The system also:
- extracts risks
- generates executive summaries
- provides negotiation recommendations

## 📊 Risk Dashboard
- Overall contract safety score
- Clause-level highlights
- Donut chart risk visualization
- Recent analysis overview

## 📑 Interactive PDF Viewer
- Clause highlights directly on rendered PDF
- Color-coded risk indicators
- Tooltip explanations

## 📥 PDF Report Generation
Download full AI-generated legal analysis reports including:
- risk breakdown
- summaries
- recommendations
- highlighted clauses

## 🕘 History System
- Save all previous analyses
- Reopen past reports instantly
- Search/filter contracts
- Delete old analyses


# 🧠 Workflow

1. User uploads contract PDF
2. Backend extracts text using PyMuPDF + OCR
3. Gemini AI analyzes legal clauses
4. Risks are classified and scored
5. Results stored in Supabase
6. Interactive dashboard + PDF report generated
7. User can revisit analysis anytime from history
